## 一、主Logo

### 方案A：军团徽章风格（推荐）

**Prompt:**
```
A professional logo design for "AI Army Identity", featuring a minimalist crest/badge symbol.
The crest combines three elements:
1. A human figure (left side, dark navy blue) and an AI circuit node (right side, neon cyan)
2. Connected by a golden thread forming an upward arrow between them
3. Below the crest, the text "AI ARMY" in bold modern sans-serif, with "IDENTITY" beneath in smaller caps

Style: Clean, corporate, trustworthy, slightly futuristic
Color palette: Deep navy (#1a1a2e), neon cyan (#00d4ff), warm gold (#ffd700)
White background, 1:1 square aspect ratio
Vector quality, suitable for GitHub README and website header
```

### 方案B：简洁科技风

**Prompt:**
```
A minimal tech logo for "AI Army Identity". 
Main symbol: A hexagon shape with a stylized human silhouette on the left half (in dark blue) 
and circuit/AI node patterns on the right half (in cyan), 
split by a diagonal golden line from bottom-left to top-right.
Below: "AI Army" in bold sans-serif, "Identity" in lighter weight.

Style: Modern, clean, Silicon Valley startup aesthetic
Colors: Dark navy blue, electric cyan, warm gold
White background, square format
```

### 方案C：纯文字标志（备选，用于小尺寸显示）

**Prompt:**
```
A typographic logo for "AI ARMY IDENTITY" in a modern, powerful style.
The word "AI" in bold cyan, "ARMY" in dark navy bold capitals beside it, 
"IDENTITY" below in smaller lighter-weight navy text.
Between AI and ARMY, a small diamond/crystal shape in gold.
Clean sans-serif font (like Inter or Montserrat).
White background, horizontal layout, suitable for GitHub header.
```

## 二、组织架构图

### 完整架构图（README首页用）

**Prompt:**
```
A professional organizational chart / architecture diagram for "AI Army Identity", 
showing a human-AI collaboration system.

Top box (largest, navy blue): "Commander (萧总)" with subtitle "Decision · Dispatch · Quality"
Below it, 6 boxes in two rows of three:
Row 1 (warm gold accent):
  "Product Center · 沈括" | "Market Center · 乔致庸" | "Tech Center · 鲁班"
Row 2 (cyan accent):  
  "Risk Center · 海瑞" | "Security Center · 小兵" | "Archive Center · 司马迁"

Each box shows: role name in bold, Chinese name below, 3 dot-point responsibilities
Connecting lines flow from Commander down through all six centers in a tree structure

Style: Clean corporate infographic, flat design
Background: White or very light gray
Box shadows: Soft, 3D-like
Color coding: Navy for command, gold for product/market/tech, cyan for risk/security/archive
High resolution, suitable for GitHub README
```

### 简化架构图（快速展示版）

**Prompt:**
```
A simple flat organizational chart. 
Top: single dark blue box "AI Army Commander" 
Below (connected by clean lines): 3 light blue boxes side by side - 
"Content Team", "Market Team", "Tech Team"
Below those: 3 light teal boxes - "Risk", "Security", "Archive"

Minimalist style, white background, clean lines, no shadows.
Modern flat design, suitable for documentation.
```
