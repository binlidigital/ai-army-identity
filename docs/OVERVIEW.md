# AI Army Identity · 系统概述

## 这是什么？

AI Army Identity 是一套**人机共生组织的操作系统**。

它不是又一个Agent框架。市面上已有的框架（AutoGPT、CrewAI、MetaGPT等）模拟的是「AI帮人干活」，而我们做的是**「人+AI组成团队一起干活」**。

## 核心差异

| 对比项 | 传统Agent框架 | AI Army Identity |
|:------:|:-------------|:-----------------|
| 团队形态 | 任务驱动的一次性协作 | **有身份、有记忆的长期组织** |
| 成员管理 | 每次重新分配角色 | **持续身份+成长记录** |
| 学习机制 | 无 | **每次纠正=进化，产出可复用skill** |
| 引擎切换 | 上下文丢失 | **身份映射+自检恢复** |
| 哲学基础 | KPI/指标驱动 | **理想驱动（Vision-Driven）** |

## 适用场景

- 想用AI组建团队的个人创作者/教育者
- 已有多个AI工具但希望统一管理的团队
- 对「AI员工长期成长」感兴趣的开发者
- 探索人机协作新范式的组织
