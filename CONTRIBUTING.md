# 贡献指南

感谢你考虑为 AI Army Identity 做贡献！

## 我们欢迎的贡献

- 🐛 Bug报告和修复
- 📖 文档改进和翻译
- 💡 新功能建议
- 🎨 设计优化
- 🤝 社区讨论

## 提交流程

1. Fork 本仓库
2. 创建你的特性分支 (`git checkout -b feature/amazing-feature`)
3. 提交你的修改 (`git commit -m 'feat: add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 创建 Pull Request

## 代码规范

- 使用语义化提交信息（Conventional Commits）
- 保持向后兼容
- 添加必要的测试和文档
- 遵循现有的代码风格

## 行为准则

我们承诺维护一个开放、友善的社区环境。详见 [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)。
